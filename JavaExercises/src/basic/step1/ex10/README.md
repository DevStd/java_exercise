## [Ex10] 표 그리기

### 1. 목표
* 표의 모서리로 사용할 문자가 `EDGE` 배열에 들어있다.
* 이 배열을 사용하여 주어진 행과 열의 수 만큼의 크기를 가지는 표를 출력하자.

```java
String[][] EDGE = {
	{"┌","┬","┐"},
	{"├","┼","┤"},
	{"└","┴","┘"}
};
```

### 2. 실행예시

* rows = 1 , cols = 1

```
┌┐
└┘
```

* rows = 2 , cols = 2

```
┌┬┐
├┼┤
└┴┘
```

* rows = 4 , cols = 6

```
┌┬┬┬┬┬┐
├┼┼┼┼┼┤
├┼┼┼┼┼┤
├┼┼┼┼┼┤
└┴┴┴┴┴┘
```
